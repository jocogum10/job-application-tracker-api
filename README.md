# Job Application Tracker - API

An app that uses React JS for the front end and Ruby on Rails API for the backend

---

## Version
 - React v3.1.2
 - Rails v7.0.4

## Setup
```
$ bundle
$ rails db:setup
$ rails s
```

## Test
```
$ rspec
```
